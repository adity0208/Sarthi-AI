**Sarthi-AI** is a multilingual, voice-activated medical assistant designed to bridge the healthcare gap in rural India. It provides non-tech-savvy users with clinical-grade triage guidance in local languages, helping distinguish between minor ailments and life-threatening emergencies.

## üåü Key Features

* **Multilingual Voice Interface**: Supports Hindi and English voice input/output for users with limited digital literacy.
* **Multi-Agent Triage**: Uses specialized AI agents (Safety, Extraction, RAG) for accurate symptom assessment.
* **Emergency Red-Flag Detection**: A dedicated Safety Agent prioritizes life-threatening symptoms and provides immediate ER guidance.
* **AWS-Powered Scalability**: Built on a serverless architecture using AWS Lambda and Amazon Bedrock for low-latency performance.
